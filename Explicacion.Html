Semántica HTML5

Etiquetas Semánticas

HTML5 introdujo etiquetas semánticas que mejoran la estructura y accesibilidad:

  • <header>: Cabecera de página o sección
  • <nav>: Contiene enlaces de navegación
  • <main>: Contenido principal único
  • <section>: Sección temática del contenido
  • <article>: Contenido independiente
  • <aside>: Contenido relacionado indirectamente
  • <footer>: Pie de página o sección

Multimedia en HTML5

Etiquetas Multimedia

HTML5 permite integrar multimedia nativamente:

  • <img>: Para imágenes 
    <img src="imagen.jpg" alt="Descripción">
  • <video>: Para videos 
    <video controls>
  <source src="video.mp4" type="video/mp4">
</video>
  • <audio>: Para audio 
    <audio controls>
  <source src="audio.mp3" type="audio/mpeg">
</audio>
  • <iframe>: Para contenido embebido 
    <iframe src="https://www.youtube.com/embed/..."></iframe>

Formularios en HTML5

Elementos de Formulario

HTML5 mejoró los formularios con nuevos tipos y atributos:

  • <input>: Con tipos como email, date, number
  • <textarea>: Para texto multilínea
  • <select>: Listas desplegables
  • <button>: Botones interactivos

Atributos importantes: required, placeholder, pattern
